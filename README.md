# tmux Cheat Sheet

A comprehensive reference guide for tmux keyboard shortcuts and commands.

## Contents

The [tmux_cheat_sheet.md](tmux_cheat_sheet.md) covers:

- **Sessions** - Creating, killing, attaching, detaching
- **Windows** - Creating, renaming, navigating, reordering
- **Panes** - Splitting, moving, resizing, zooming
- **Copy Mode** - Vi keybindings, searching, copying, pasting
- **Miscellaneous** - Command mode, global options

## Quick Reference

| Action | Shortcut |
|--------|----------|
| New session | `tmux new -s name` |
| Detach | `Ctrl+b d` |
| Attach | `tmux a -t name` |
| Split vertical | `Ctrl+b %` |
| Split horizontal | `Ctrl+b "` |
| Next pane | `Ctrl+b o` |
| Zoom pane | `Ctrl+b z` |
| New window | `Ctrl+b c` |
| Next window | `Ctrl+b n` |
| Copy mode | `Ctrl+b [` |
